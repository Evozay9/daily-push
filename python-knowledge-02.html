01 · 变量：给数据起名字

变量就是一个「贴了标签的盒子」

想象你有很多盒子，每个盒子里装着一个东西。变量就是这些盒子的标签。

在Python里，变量名 = 值，等号不是「等于」，是「把右边的值放到左边的盒子里」。

name = "小明" age = 18 height = 1.75 is_student = True

上面创建了4个变量：名字是字符串，年龄是整数，身高是小数，是否学生是布尔值。

之后你可以随时用这个名字取出里面的值，也可以换成新值。

02 · 四种基本数据类型

Python盒子里可以装四种基本东西

int（整数）：1、-5、1000000，没有小数点的数字。

float（浮点数）：3.14、-0.5、1.0，有小数点的数字。

str（字符串）：用引号包住的文字，"hello"、'你好'、"123"（注意这是文字不是数字）。

bool（布尔值）：只有True和False两个值，代表「是」或「否」。

数据类型像食材的包装形式。
同样是「3」，整数3可以直接加减，字符串"3"只能拼接文字。
就像液体装在瓶子里，固体装在袋子里——形式不同，用法就不同。
03 · 变量的命名规则

好的变量名让代码像说话一样

✅ 可以用：字母、数字、下划线，不能以数字开头

✅ 推荐：user_name、total_price、is_valid

❌ 不能用：2name（数字开头）、my-name（连字符）、for（关键字）

好的命名是：看名字就知道装的是什么。x = 18 不如 user_age = 18。
